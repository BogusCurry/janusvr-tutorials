Where did we get those numbers for the object positions, you ask? With math, of course! We want to organize everything in a circle around the center of the room, with a nice comfortable 3m radius.

A circle is 360 degrees, aka 2*PI radians, and we have 9 objects to place in a circle. So by dividing the size of the circle by the number of items, we find that we should place the item every 40 degrees, or 0.7 radians.

We can then use some basic trigonometry to turn those into (x,y,z) coordinates for each item: 

var numItems = 9,
    radius = 3,
    stepSize = 2 * Math.PI / numItems;

for (var i = 0; i < numItems; i++) {
  var x = Math.sin(i * stepSize) * radius,
      y = 1,
      z = Math.cos(i * stepSize) * radius;
  console.log(i, x.toFixed(1) + ' ' + y + ' ' + z.toFixed(1))
}
